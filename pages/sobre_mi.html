Todos los cursos realizados

Actualmente me encuentro cursando la carrera de Ciencia de Datos en Coderhouse, donde estoy adquiriendo conocimientos y habilidades para desarrollar proyectos de análisis de datos de principio a fin. Estoy aprendiendo a recolectar, limpiar y manipular datos, así como a crear visualizaciones efectivas y tableros de control interactivos. Utilizo herramientas como Python, Pandas, SQL y Power BI, y me estoy formando en técnicas estadísticas y de machine learning para transformar datos en información útil para la toma de decisiones. Me apasiona descubrir insights y generar valor a través del análisis de datos.

Esta carrera en Coderhouse estan dovididas en 4 diferentes cursos

Data Analytics

Como parte de esta formación, ya completé y aprobé el curso de Análisis de Datos, donde profundicé en el manejo de bases de datos relacionales, consultas SQL y el desarrollo de tableros dinámicos con Power BI. Aprendí a aplicar funciones DAX, Lenguaje M y Power Query para transformar y visualizar datos de forma estratégica, lo que me permite abordar proyectos de análisis de manera integral, desde la exploración inicial hasta la presentación final de resultados.

Data Science I: Fundamentos para la Ciencia de Datos

También completé y aprobé el curso Data Science I: Fundamentos para la Ciencia de Datos, donde incorporé herramientas clave para abordar proyectos de ciencia de datos desde una perspectiva técnica y analítica. Aprendí a explorar datos con Python y librerías estadísticas, comprendí el ciclo de vida de un proyecto de Data Science y desarrollé habilidades para seleccionar variables relevantes (feature selection), aplicar algoritmos, calcular métricas y validar hipótesis. Además, profundicé en técnicas de visualización para comunicar hallazgos de forma efectiva y conectar los análisis con problemáticas reales de negocio.

Data Science II: Machine Learning para la Ciencia de Datos

Actualmente me encuentro cursando el módulo Data Science II: Machine Learning para la Ciencia de Datos, donde estoy profundizando en técnicas de adquisición, manipulación y análisis exploratorio de datos (EDA) con herramientas como Numpy y Pandas. En esta etapa estoy aprendiendo a desarrollar proyectos completos de ciencia de datos, incluyendo la construcción, entrenamiento y validación de modelos de Machine Learning. También estoy incorporando criterios para seleccionar el mejor modelo según métricas específicas y la naturaleza del problema, con el objetivo de generar insights valiosos y aplicables a problemáticas reales de negocio.
